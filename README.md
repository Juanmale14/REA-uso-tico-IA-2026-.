## 🛠️ ¿Cómo adaptar o reutilizar este REA?
Este recurso está pensado por y para la comunidad educativa. Si deseas modificarlo para adaptarlo a tu aula, solo debes:

1.  **Descargar el archivo fuente:** Descarga el archivo con extensión `.elpx` guardado en este repositorio.
2.  **Abrir con eXeLearning:** Abre la aplicación [eXeLearning](https://exelearning.net/) (versión 4.0 o superior) y carga el archivo.
3.  **Modificar y personalizar:** Adapta los dilemas éticos, añade nuevos ejemplos o actividades según el nivel de tus alumnos.
4.  **Publicar de nuevo:** Exporta el resultado como "Sitio web" y súbelo a tu propio GitHub, blog o plataforma educativa (Moodle, Google Classroom, etc.).
